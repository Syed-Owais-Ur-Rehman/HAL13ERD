# HAL13ERD

<br>

## Description

HAL13ERD is a Kali virtual machine created with the intent to gather various OSINT tools to perform basic OSINT investigations. This project's objective is to replicate the once-known Buscador Virtual Machine by creating your own version.

## OSINT Tools

The following tools used in the project are listed below:
* Sherlock
* TheHarvester
* GHunt
* Social_mapper
* EXIF Tool
* PhoneInfoga
* Osintgram
